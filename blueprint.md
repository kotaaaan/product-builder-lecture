# Lotto Number Generator

## Overview

This is a simple web application that generates and displays a set of 6 random lottery numbers.

## Features

*   Generates 6 unique random numbers between 1 and 45.
*   Displays the numbers in a visually appealing way.
*   Allows the user to generate a new set of numbers with a button click.
*   **Modern and visually appealing design with animations.**
*   **Readable and clean typography using web fonts.**

## Design

*   **Layout:** Centered layout with a title, number display, and a "Generate" button.
*   **Colors:** A vibrant color palette using modern color spaces (OKLCH).
*   **Typography:** Clear and readable fonts from Google Fonts.
*   **Effects:** Gradient background, shadows, and animations.
